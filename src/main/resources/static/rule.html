Règles du jeu

Bienvenue dans les règles de Nin-Jardins:

  • Le but du jeu est de capturer le plus de pièces possibles.
  • Le pion noir commence toujours la partie. Puis, les joueurs s'alternent pour placer leurs pions en capturant des pions adverses durant leur tour.
  • Pour capturer les pions adverses, vous devez entourer une ou plusieurs de leurs pièces entre deux de vos pièces en ligne, colonne ou diagonale. Les pions entourés sont alors retournés et deviennent les vôtres.
  • Le jeu se termine lorsque le plateau est plein ou qu'aucun joueur ne peut placer de pion. Le joueur avec le plus de pions sur le plateau gagne.
